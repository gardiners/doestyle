# Colours provided by `doestyle`

A dataset containing names, hex and RGB values, and other identifiers
for each colour provided in `doestyle`. Its content can be visualised
with
[`show_colours()`](https://nsw-education.github.io/doestyle/reference/show_colours.md)
or further manipulated.

## Usage

``` r
colourdata
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 51
rows and 13 columns.
